KADATH

A multiplayer game engine.

KADATH is a project for developing a multiplayer game engine. The client and the server code are being released under the GPL.

The client side of the engine was being developed under OGRE (check ngranek.com and my blog (in spanish) for further information on the previous versions), but a bug in the ATI drivers for Linux and a problem with the PagingLandscape plugin have halted the development of that version (we may restart in the future). This version uses the impressive jME engine.

Right now we are porting everything from the OGRE version to this version, so a lot of changes and refactoring is going to take place.
